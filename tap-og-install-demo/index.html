Tanzu Platform

From Install to Demo

Objective

Starting with a 'fresh' k8s cluster...

Deploy and configure TAP (via GitOps) then use it to...

Deploy microservices with backing database and front-end api management

At end of demo I can...

Deploy my app jar/binary in one command

Create a cache, database, message broker in one command

'Bind' my app to the service instance

Declaratively 'protect' my API with a platform gateway

Manage api-key based access from a mgmt portal

Prerequisites

"Fresh" K8s Cluster

DNS server access with wildcard dns domain certs

GitHub client-id & client-secret

GitHub access token

'age'key encryption key (create with age-keygen)

Image registry - eg Harbor

IDP account creds (eg Okta, for spring api demos)
